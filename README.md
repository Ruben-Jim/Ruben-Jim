Founder of **[CodeWithRuben](https://github.com/CodeWithRuben)** — a dev studio that helps local businesses go from "taking bookings by text" to running their entire operation through a custom-branded app.

## 🚀 Featured Projects

### 🧹 [ProCleaning Seattle](https://tradeservice.expo.app)
**Expo · React Native · Firebase · Stripe**

Full operations platform for a Seattle cleaning company — customers book, pay, and track jobs while the owner manages everything from one dashboard.

- 📅 Online booking + subscription management
- 💳 Stripe payments + recurring billing
- 👷 Worker management + job tracking
- 📄 Automated invoice + estimate generation
- 📊 Admin dashboard with real-time order management
- 🔔 Push notifications for bookings and updates

---

### 🏘️ [Shelton Springs HOA](https://hoa-demo--l91yvra8kn.expo.app)
**Expo · React Native · Firebase · Stripe**

Community management platform for HOA residents and administrators.

- 📢 Posts, announcements, and community polls
- 💰 Stripe payment processing for HOA fees and fines
- 🔐 Role-based access — residents vs admin
- 📱 iOS + Android + web from one codebase

---

### 💇 [RosaSalon](https://beautysalontemplate.expo.app)
**Expo · React Native · Firebase · NativeWind**

Bilingual booking platform for a beauty salon — clients book appointments, owners manage schedules.

- 📆 Real-time appointment calendar
- 🔔 Automated reminders — reduces no-shows
- 👩‍💼 Multi-stylist scheduling
- 🌎 English + Spanish
